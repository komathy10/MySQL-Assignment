This project is a simple MySQL database system built using an online compiler (Newton School).
It includes operations for managing customer details and product details, along with various SQL queries demonstrating:

->Table creation
->Data insertion
->Aggregate functions
->Logical operators (AND / OR)
-?Sorting using ORDER BY
->Basic data retrieval
